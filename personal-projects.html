Página personal

Repositorio en GitHub

Una página web hecha con Sass (usando Live Sass, plugin de VS Code), HTML y una pequeña cantidad de vanilla Javascript, sin template engines, librerías ni frameworks de ningún tipo.

El objetivo de éste proyecto es practicar mis habilidades de maquetado (ya que considero que es uno de mis eslabones débiles, junto a DevOps) y, en el proceso, disponer de un portafolio y de un currículum online personal. Es por eso que he priorizado no usar frameworks de estilo como Bootstrap o Bulma.

Stack tecnológico

  • HTML

  • Sass

  • Vanilla Javascript

  • GitHub

  • Netlify
