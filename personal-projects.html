Página personal

Repositorio en GitHub

Una página web hecha con Sass (usando Live Sass, plugin de VS Code), HTML y una pequeña cantidad de vanilla Javascript, sin template engines, librerías ni frameworks de ningún tipo.

El objetivo de éste proyecto es practicar mis habilidades de maquetado (ya que considero que es uno de mis eslabones débiles, junto a DevOps) y, en el proceso, disponer de un portafolio y de un currículum online personal. Es por eso que he priorizado no usar frameworks de estilo como Bootstrap o Bulma.

Stack tecnológico

  • HTML

  • Vanilla Javascript

  • Sass

  • GitHub

  • Netlify

Reservation maker (Work in progress)

Repositorio en GitHub

Una API REST escrita en Java 8 que permite realizar acciones CRUD para gestionar un sistema de reservas, realizado con TDD.

Ésta está creada con SparkJava, un microframework de Java que permite crear microservicios o webapps, teniendo como base una serie de utilidades que se asemejan a un request dispatcher puro y duro.

El punto de usar SparkJava en lugar de Spring Boot es demostrar que, pese a que Spring Boot a nivel corporativo es muy usado y te da muchas facilidades, estoy capacitado para crear aplicaciones backend con otros frameworks.

Ésta aplicación incluye:

  • Inyección de dependencias (Google Guice)
  • Autenticación y autorización (JWT)
  • [WIP] Docker y Docker compose
  • [WIP] Migraciones y seeders para BBDD

Stack tecnológico

  • Java 8

  • SparkJava

  • MySQL

  • Maven

  • Docker

  • JUnit 4

  • Mockito
